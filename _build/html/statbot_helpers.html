statbot_helpers.check_changes module¶

statbot_helpers.check_changes.check_changes(fname)¶

Checks if change in input file occured. If a change occured in known file the corresponding hash in data/hashes.csv will be updated and it will return True. If file not known it will add name of file and corresponding hash to data/hashes.csv

Parameters

fname (string) – Path of file to be checked

Returns

True if change in file occured, False if no change occured or file unknown

Return type

bool
